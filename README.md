# 🪸 CoralVision – AI-Based Coral Reef Health Detection

## 🚀 Overview

CoralVision is a deep learning web app that classifies coral reef images as **Healthy** or **Bleached** using EfficientNet-B0.

## 🧠 Tech Stack

Python, PyTorch, Flask, Torchvision, PIL

## ⚙️ Features

* Real-time coral health detection
* 95%+ accuracy
* Image upload and instant prediction
* Confidence score display
* Interactive web interface

## ▶️ Run Locally

```bash
pip install -r requirements.txt
python app.py
```

Open:
http://127.0.0.1:5000
